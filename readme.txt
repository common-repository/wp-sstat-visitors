== Description ==

Track your blog visitors path. Find what was the visitors path 
and timings in your site. This Plugin present some information 
on each visitor at your site, including where did he came from, 
what was he doing and for how long.

WP-sstat-visitors plugin is an extension for [WP-shortstat(2)](http://wordpress.org/extend/plugins/wp-shortstat2/ "REquired plugin") plugin.

== Frequently Asked Questions ==

= Q: When I click on Visitors, your plugin told me that I have to install WP-Shortstats. But it's allready installed. =
The plugin assume the presence of a table named {prefix}ss_stats, created by WP-Shortstat. Please check that you have the assumed table name.

For any other questions please [contact my](http://mytecblog.wordpress.com/wp-sstat-visitors/ "plugin home page") or leave a comment with any question.

== Installation ==

WP-sstat-visitors plugin is an extension for [WP-shortstat(2)](http://wordpress.org/extend/plugins/wp-shortstat2/ "REquired plugin") plugin and dependent on it.
Install [WP-shortstat(2)](http://wordpress.org/extend/plugins/wp-shortstat2/ "REquired plugin") first and then install WP-sstat-visitors.

1. Install the plugin through the 'Plugins|Add New' menu in WordPress or manually download and extract WP-sstat-visitors.zip to the /wp-content/plugins/ directory. 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now you can see your visitors in DashBoard|visitors.
